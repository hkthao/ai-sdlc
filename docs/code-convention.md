# Code Convention

General:
- Use TypeScript strictly
- No `any`
- Prefer const over let

Vue:
- Composition API only
- One component per file
- Props must be typed
- Emit events explicitly

Naming:
- Components: PascalCase
- Variables: camelCase
- Files: kebab-case or PascalCase (choose one)

Testing:
- One test file per component
- Test file name: <ComponentName>.spec.ts
